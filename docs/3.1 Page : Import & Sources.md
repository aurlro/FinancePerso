Page : Import & Sources

Objectif :
Permettre l’import de relevés bancaires personnels (CSV / Excel) de manière simple et fiable.

Contenu :
- Zone d’upload fichiers (drag & drop)
- Sélecteur de type de compte :
  • Compte personnel
  • Compte joint
- Tableau de prévisualisation des transactions importées
- Indication claire du statut (succès / erreur)
- Bouton principal : “Importer et analyser”

Style :
- Sobre, clair
- Palette “Finance sereine”
- Accent discret uniquement pour les messages importants

UX :
- Rassurant (données privées, usage local)
- Peu d’éléments à l’écran
