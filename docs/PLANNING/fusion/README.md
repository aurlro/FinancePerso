# Projet Fusion - FinancePerso

> Application de gestion financière pour couples et familles

---

## 🎯 Vision

Fusion est une application complète de gestion financière personnelle dédiée aux couples et familles. Elle permet:

- **Gestion partagée** des finances en couple
- **Suivi individuel** et collectif des dépenses
- **Budgets collaboratifs** avec objectifs communs
- **Transparence** et communication financière
