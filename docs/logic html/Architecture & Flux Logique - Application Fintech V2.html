2. Exemples de Parcours Utilisateur (Use Cases)

Exemple 1 : L'Ingestion Intelligente

Événement : L'utilisateur upload son relevé bancaire (CSV) via app.py.

1. database_manager.py lit le fichier et le convertit en DataFrame standardisé.

2. processor.py prend le relais : la fonction Merchant Cleaning détecte "PRLV SEPA NETFLIX".

3. Le modèle catégorise automatiquement : Abonnement > Divertissement.

Résultat UI : Le Dashboard (app.py) met à jour le graphique "Cash-Burn" et isole cette dépense dans la jauge des charges fixes.

Exemple 2 : Le Scénario "What-If" (IA)

Événement : L'utilisateur utilise le slider dans app.py : "Réduire mon budget Loisirs de 15%".

1. La nouvelle valeur est envoyée à processor.py.

2. Le module de calcul réévalue le "Reste à vivre" projeté et génère un prompt dynamique pour le LLM.

3. Le LLM répond : "Avec cet effort, votre objectif de 5000€ pour le Japon sera atteint 2 mois plus tôt."

Résultat UI : Un rapport texte stylisé en Markdown s'affiche sous le graphique d'impact.
