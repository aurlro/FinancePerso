Exemples Pratiques du Fonctionnement (Logique métier)

💡

Exemple 1 : Le flux "Active Learning"

Objectif : Comprendre comment une correction de l'utilisateur améliore l'IA pour l'avenir.

  • 1. L'utilisateur voit "Uber Eats" classé dans Transport dans le st.data_editor (Module 3A).
  • 2. Il modifie la cellule en Alimentation > Livraison.
  • 3. L'événement est capturé par app.py, qui appelle la fonction update_dictionary() dans database_manager.py (Module 1).
  • 4. Le processor.py (Module 2A) met à jour son dictionnaire Regex local. Au prochain import, tous les "Uber Eats" seront correctement catégorisés sans appel à l'API LLM (gain de temps et d'argent).
🧠

Exemple 2 : Le flux "Conseil d'Épargne"

Objectif : Voir comment les 3 modules collaborent pour générer un conseil personnalisé.

  • 1. Le processor.py (Module 2B) calcule que le "Reste à vivre" actuel est de 450€.
  • 2. L'utilisateur bouge le st.slider "Loisirs" (Module 3C) pour simuler une baisse de 50€.
  • 3. app.py envoie cette variable (Reste à vivre projeté = 500€) au module GenAI (Module 2C).
  • 4. processor.py formate le prompt : "L'utilisateur épargne 500€. Rédige un encouragement de 2 lignes." Le LLM répond et app.py affiche le texte via st.success().
